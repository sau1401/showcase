Rust can use cargo. which is another way to deal with Rust.
  • create a cargo w/ "cargo new [name]"
  • it creates two files, a rust file and a toml file
  • cargo had generated code into the source directory... which organizes your files and other projects accordingly

Oddities with booleans and 0/1:

 
                        fn main() {
                           let number = 3; //this doesn't automatically see it as a boolean (i.e 0 or 1)... it needs to be set as a boolean (true or false) to work

                           if number { //otherwise we need to do "if number != 3" or whatever
                              println!("number was three");
                          }
                       }

Using let w/ if:

 
                        fn main() {
                          let condition = true; //set the condition to true
                          let number = if condition { 5 } else { 6 }; //and if the condition is TRUE, set number to 5 otherwise set it to 6

                          println!("The value of number is: {number}");
                       }

Using "loop" to constantly repeat code:

 
                            fn main() {
                                loop {
                                    println!("again!"); //will keep printing "again" until you stop it w/ ctrl+c
                                }
                            }
